2. Описание сервиса

Бот предоставляет услуги генерации изображений и видео с помощью нейросетей:

  • Генерация изображений по текстовому описанию (промпту)
  • Генерация изображений на основе референсных фотографий
  • Генерация видео по текстовому описанию
  • Перенос движения с видео на изображение
