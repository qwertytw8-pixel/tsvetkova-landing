## Что нужно сделать далее

### 1. Telegram-бот для обратной связи
Форма "Свяжитесь с нами" должна отправлять уведомления в Telegram.

**Что нужно:**
- Создать Telegram-бота через @BotFather → получить токен
- Узнать Chat ID 2-3 админов (Екатерина с 2 аккаунтов)
- Создать Cloudflare Worker (бесплатно), который:
  - Принимает POST-запрос с данными формы (имя, контакт, сообщение)
  - Отправляет сообщение через Telegram Bot API (`/sendMessage`) каждому админу
- В `index.html` подключить отправку формы через `fetch()` на URL воркера

**Пример Cloudflare Worker:**
```javascript
export default {
  async fetch(request, env) {
    if (request.method !== 'POST') return new Response('Method not allowed', { status: 405 });
    const data = await request.json();
    const text = `Новая заявка!\nИмя: ${data.name}\nКонтакт: ${data.contact}\nСообщение: ${data.message}`;
    const chatIds = [env.CHAT_ID_1, env.CHAT_ID_2]; // из env переменных воркера
    for (const chatId of chatIds) {
      await fetch(`https://api.telegram.org/bot${env.BOT_TOKEN}/sendMessage`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ chat_id: chatId, text, parse_mode: 'HTML' })
      });
    }
    return new Response(JSON.stringify({ ok: true }), { headers: { 'Content-Type': 'application/json', 'Access-Control-Allow-Origin': '*' }});
  }
}
```

### 2. Кастомный домен
GitHub Pages поддерживает кастомные домены:
1. Купить домен (например, tsvetkovaproai.ru)
2. В настройках DNS добавить CNAME-запись: `www` → `qwertytw8-pixel.github.io`
3. Или A-записи на IP GitHub Pages: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
4. В Settings → Pages → Custom domain указать домен
5. Включить "Enforce HTTPS"
